## Introduction

基于 TensorRT 的 C++ 推理库，支持 Yolo 系列 V5、X、V7、V8 目标检测算法，STARK、OSTrack 单目标跟踪算法，Bytetrack、Deepsort 多目标跟踪算法，未来还将支持更多检测、跟踪、分割模型。

未来计划支持 Openvino 和 rknn 框架的 C++ 推理。
